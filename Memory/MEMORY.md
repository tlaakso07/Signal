## Skills Stack — The 7-Layer Execution Engine (Installed April 21, 2026)

All skills live in `Skills/`. All CLIs live in `CLI/`. These are not optional enhancements — they are the pipeline. Load the relevant layer before any UI, design, or animation work.

| Layer | Skill | What It Does |
|---|---|---|
| 1 — Intelligence Extraction | **SkillUI** `Skills/npxskillui/` | Reverse-engineers any live site into a Claude-ready design system. Ultra mode = full visual extraction. Point at B&B + Uncle Ike's first. |
| 2 — Design System Generation | **UI UX Pro Max** `Skills/ui-ux-pro-max-skill/` | 161 reasoning rules + 67 UI styles + Design System Generator. Feed the Identity File → get full palette, type, pattern, effects, QA checklist. SkillUI = current state. This = ideal state. |
| 3 — Build Intelligence | **Taste Skill** `Skills/taste-skill/` | 8 frontend variants + 3-dial tuning: DESIGN_VARIANCE / MOTION_INTENSITY / VISUAL_DENSITY (1–10). For B&B: `taste-skill`, high variance, moderate motion. |
| 4 — Reference Library | **Awesome DESIGN.md** `Skills/awesome-design-md/` | 69 production DESIGN.md files from real sites. Signal never builds from zero — we build from proof. |
| 5 — Animation & Motion | **WebGPU Three.js TSL** `Skills/webgpu-claude-skill/` | GPU shaders, particle systems, cinematic post-processing. The Anti-Gravity Terminal engine. They export a PNG. We render a living brand experience. |
| 6 — Quality Control | **Impeccable** `Skills/impeccable/` | 18 commands. Anti-pattern enforcement. The final gate — nothing ships without `/critique` + `/polish`. Zero exceptions. |
| 7 — Live Verification | **Playwright CLI** `CLI/playwright-cli/` | Token-efficient browser automation. Navigate, click, screenshot, test. Every deployed deliverable gets verified and screenshotted as proof. |
